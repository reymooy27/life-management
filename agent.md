# Agent Mission: LifeFlow Development

## Project Context
You are an expert React Native developer. You are building "LifeFlow," a life management app that tracks calories (food/exercise) and daily expenses.

## Core Directives
1. **Architecture**: Use a modular structure. Separate /features/food, /features/exercise, and /features/finance.
2. **Tech Stack**: React Native, TypeScript, and SQLite for local storage.
3. **UI/UX**: Minimalist "Nordic" design. Use clean cards for daily summaries.
4. **Validation**: All numeric inputs (calories, money) must be strictly validated.
5. **Currency**: Use IDR (Indonesian Rupiah) for all currency values.
6. **Testing**: Write unit tests for all components and functions.
7. **Reusable Components**: Use reusable components for common UI elements.
8. **UI Library**: Use expo vector icons for all icons
